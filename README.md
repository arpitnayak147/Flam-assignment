# AI R&D Assignment — Parametric Curve Fitting

## Problem

The objective of this assignment is to estimate the unknown parameters **θ (theta)**, **M**, and **X** of the following parametric curve using only the provided coordinate samples in `xy_data.csv`.

\[
x(t)=t\cos(\theta)-e^{M|t|}\sin(0.3t)\sin(\theta)+X
\]

\[
y(t)=42+t\sin(\theta)+e^{M|t|}\sin(0.3t)\cos(\theta)
\]

Subject to the following constraints:

```
0° < θ < 50°
-0.05 < M < 0.05
0 < X < 100
6 < t < 60
```

---

# Objective

The dataset contains only the observed `(x, y)` coordinates and does not provide the corresponding parameter value `t` for each point. Therefore, the objective is to recover the unknown parameters `(θ, M, X)` by finding the parametric curve that best matches the observed point cloud.

---

# Approach

Since the dataset does not contain the parameter `t` for each observation, a direct nonlinear regression cannot be applied. Instead, the problem is formulated as a **point-cloud-to-curve optimization** problem.

The complete optimization procedure consists of the following steps.

## 1. Candidate Curve Generation

For every candidate parameter set `(θ, M, X)`, the parametric curve is uniformly sampled over

```
6 ≤ t ≤ 60
```

to generate a dense approximation of the predicted curve.

---

## 2. Nearest Point Matching

Each observed point from `xy_data.csv` is compared with every sampled point on the predicted curve.

The nearest point is selected, eliminating the need to know the original parameter `t` associated with each observation.

---

## 3. Objective Function

The optimization minimizes the sum of squared nearest-point distances.

\[
J(\theta,M,X)
=
\sum_{i=1}^{N}
\min_t
\left[
(x_i-\hat{x}(t))^2
+
(y_i-\hat{y}(t))^2
\right]
\]

where

- \(P_i=(x_i,y_i)\) represents an observed point.
- \(\hat{x}(t),\hat{y}(t)\) are points generated by the parametric curve.
- The minimum is evaluated over all sampled values of \(t\).

A lower objective value indicates a better agreement between the predicted curve and the observed dataset.

---

## 4. Global Optimization

The optimization begins with **Differential Evolution**.

Differential Evolution performs a global search over the complete parameter space while respecting the assignment bounds.

This significantly reduces the probability of converging to poor local minima.

---

## 5. Local Refinement

The solution obtained from Differential Evolution is further refined using the **Nelder–Mead** optimization algorithm.

A much denser sampling of the curve is used during this stage to improve numerical accuracy and obtain the final parameter estimates.

The recovered parameters produce an excellent agreement with the provided dataset.

| Metric | Value |

The residual values are close to zero, indicating that the estimated curve closely matches the observed data.
